
# Clash Of OOP a ruby game

## Purpose
This repository presents an Object Oriented Programming game in Ruby. 
You'll find :
* version 1.0
* version 2.0
* version 3.0

## Installation
**clone** the repo first
to install needed gemfiles :
```bundle install ```

## How to Use This
Launch the ruby files from the root directory.
```ruby app.rb```
...and you'll be able to enjoy the gameplay in the terminal!
